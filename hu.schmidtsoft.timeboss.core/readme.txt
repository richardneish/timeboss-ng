Libraries used by Timeboss are under different open source licenses.
Libraries used and packed with Timeboss:
 - jaxen
 - dom4j

directory structure of distribution package:
 /lib - dependent 3rd party libraries
 /src - source code of timeboss
 /readme.txt - this file
 /gpl-v3.txt - GPL v3 license text
 /timeboss.jar - java executable archive that contains timeboss and all dependent libraries
 
 Usage:
 Start timeboss with the following command line:
 java -jar timeboss.jar
